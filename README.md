# gopen-port

Simple port scanner in Go.


# Usage
```
Usage of ./gopen-port:
  -host string
    	Host to scan
  -max int
    	The maximum port you'd like to scan (default 1024)
  -min int
    	The minimum port you'd like to scan (can't be bellow 0)
  -timeout int
    	Timeout in millisseconds (default 1000)
```
